# MediLink – Health & Wellness Platform

## Overview
This project was developed as part of the PIDEV – 3rd Year Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).

MediLink is a web platform that connects patients with healthcare professionals. It allows users to book appointments, manage medical information, and improve healthcare accessibility.

## Features

- User registration and authentication
- Appointment booking system
- Doctor availability management
- Admin dashboard
- Secure user management
- Health and wellness services

## Tech Stack

### Frontend
- HTML
- CSS
- Bootstrap
- Twig

### Backend
- Symfony 6.4
- PHP
- Doctrine ORM
- MySQL

## Architecture
The application follows the MVC architecture implemented by Symfony. The system uses Doctrine ORM for database management and Twig templates for rendering the frontend.
